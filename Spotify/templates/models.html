Testing Machine Learning Models

Model #1: Decision Tree


With Pandas, we pulled data from Zoe's, Mamta's and Kelly's Spotify playlists and sorted the outcomes as integers; 0 for Zoe, 1 for Mamta and 2 for Kelly. We ran a tree decision classifier algorithm on the data using SciKitLearn and visualized the classification with Graphviz. The model scored 0.6 on the test data which suggests that the tree decision classifier was not the best machine learning algorithm as it overfits our data. An advantage of this algorithm is that we can understand the features that reflect best for each user.



Model #2: OneClass SVM


Similar to the previous model, we used Pandas to pull playlist and song information for Zoe, Kelly, and Mamta. We then used the SciKitLean library to use the OneClassSVM algorithm to create three separate models for Zoe, Kelly, and Mamta. The purpose of this was to see if we tested a different playlist, we should be able to predict if any of the three of us would enjoy listening to it. We did run into an issue where we needed many more songs for each of us to test on because of now it seems to identify all songs outside of those in our dataframes as outliers. This model could potentially be good for mood playlist predicting.


Selected Model:


For the above reasons, we chose to pursue the Tree Decision model in the hopes of predicting Spotify users' musical interest.
